A very basic and simple First Person Shooter Engine inspired by javidx9's Console FPS

## How to build
  ```bash
  git clone https://github.com/miha53cevic/SimpleFPS.git
  cd SimpleFPS
  cmake .
  make
  ```
You should now have a bin folder with an executable
